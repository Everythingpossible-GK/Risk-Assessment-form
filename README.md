This interactive web-based tool is designed to streamline the creation, management, and export of professional Risk Assessment forms. It utilizes a standard 5x5 Risk Matrix to quantify workplace hazards and manage action plans.

🛠 Features

Bilingual Support: Toggle between English and Arabic interfaces instantly.

5x5 Risk Matrix: Interactive modal to calculate risk scores ($Likelihood \times Consequence = Rating$).

Action Plan Management: Track actions with dedicated statuses (Not Started, In Progress, Completed, Cancelled).

Data Persistence: Save your assessment progress as a .json file and restore it later to continue working.

Professional Exports: Generate formatted .xls (Excel), .doc (Word), or high-quality landscape printouts.

📊 Risk Matrix Methodology

The tool uses a standard scoring system to determine the priority of control measures:

🚀 Getting Started

Fill Header Info: Enter the Owner, Staff ID, Department, and Assessment dates.

Identify Hazards: Add rows for each identified hazard.

Calculate Risk: Click on the $L$ or $C$ cells to open the Risk Matrix and select the appropriate score.

Define Actions: Assign owners and deadlines for additional controls.

Save/Export: Use the toolbar to save your progress or print the final document in A4 Landscape.

This tool is a client-side application.
