Reporting a Vulnerability

We take the security of this tool seriously. If you discover a vulnerability, please follow these steps:

Do not disclose publicly: Please do not open a public GitHub Issue for security vulnerabilities.

Report via Email: Send a detailed report to security@example.com (replace with your actual contact email).

Response Time: You can expect a response within 48 hours. We will provide a timeline for the fix and keep you updated.

Data Privacy & Security

The Risk Assessment Tool is a serverless, client-side application:

No Data Collection: We do not collect, store, or transmit any data entered into the forms.

Local Storage: All data exists only in the volatile memory of your browser.

JSON Security: When using the "Restore Data" feature, ensure you are only uploading .json files generated by this tool. Do not upload files from untrusted sources.

Best Practices

Always use the "Save Data (JSON)" feature frequently to prevent data loss due to browser refresh.

Review exported Word and Excel documents to ensure data integrity before final submission.
